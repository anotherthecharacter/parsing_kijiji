<h2 align="center">kijiji.ca apartments parsing</h2>

### Инструменты разработки

**Стек:**
- Python >= 3.10
- BeautifulSoup4
- Postgresql
- SQLAlchemy

## Старт

#### 1) Установить зависимости

    pip install -r requirements.txt

##### 2) Запустить парсинг

     python3 main.py

##### 3) Настроить отображение в файле view.py


##### 4) Вывести результаты

    python3 view.py
